# Images Folder

Place your village photos here and update the references in `index.html`:

| Placeholder in code      | Replace with                          |
|--------------------------|---------------------------------------|
| `VILLAGE_IMG`            | Village / hero background photo       |
| `SARPANCH_IMG`           | Sarpanch (President) portrait photo   |

**Supported formats:** `.jpg`, `.jpeg`, `.png`, `.webp`

**Example usage in index.html:**
```html
<!-- Hero background -->
<div class="hero-bg" style="background-image:url('assets/images/village.jpg')">

<!-- Sarpanch photo -->
<img src="assets/images/sarpanch.jpg" alt="Sarpanch">
```
